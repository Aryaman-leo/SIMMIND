Return JSON only." def _get_cache_key(agent: Agent, idea: str) -> str # Redis key: hash(archetype+conviction_band+idea[:50]) # Agents with same archetype+conviction band get cached response
class SimulationEngine (engine/simulation.py)
class SimulationEngine: # Main tick loop — orchestrates everything def run(config: SimulationConfig) -> AsyncGenerator[TickSnapshot] # Yields a TickSnapshot after each tick for WebSocket streaming def _build_graph(agents, network_type) -> nx.Graph # NetworkX: small_world=watts_strogatz, scale_free=barabasi_albert def _pick_pairs(graph, agents, n_pairs=10) -> list[tuple] # Picks 10 random connected pairs per tick # Weighted: higher-conviction agents picked more often (influencer effect) async def _run_tick(pairs, idea, mind_engine) -> list[MindOutput] # Runs all pairs concurrently with asyncio.gather() # Updates agent states, saves snapshot to SQLite def _seed_idea(agents, idea, seed_count=3) # Sets 3 early_adopter agents to stage=aware, conviction=0.6 # These are tick 0 — the founder's first customers/contacts

Returns ranked list.
prompt Persona generation prompt
SYSTEM: You are a market research expert. Generate realistic buyer personas. Return ONLY a JSON array. No preamble, no markdown. HUMAN: Generate {n} diverse buyer personas for this target market: {market} Product they will evaluate: {brief} Price point: {price} Each persona must have: - name, age, job_title, company_size - archetype: one of [early_adopter, skeptic, conformist, influencer, analyst] - tech_comfort: float 0.0-1.0 - risk_tolerance: float 0.0-1.0 - budget_authority: bool - current_solution: string (what they use today) - switching_cost: low|medium|high - personality_summary: 1 sentence Ensure realistic distribution. Return JSON array only.
prompt Agent reasoning prompt (called every interaction)
SYSTEM: You are {name}, {age} years old, {job_title}. Personality: {personality_summary} Tech comfort: {tech_comfort}/1.0. Risk tolerance: {risk_tolerance}/1.0 You currently use: {current_solution} Your switching cost is {switching_cost}. Recent memory: {memory[-3:]} Respond as this person would — with their doubts, habits, and priorities. Return ONLY valid JSON. No other text. HUMAN: {speaker_name} just told you about a product: "{idea_version}" Price: {price_point} Reason through this honestly as {name}: 1. Do I actually have this problem? 2. Is this solution believable? 3. Would I pay {price_point} for this? 4. What is my biggest objection right now? 5. Would I mention this to someone else? Return JSON with exactly these keys: { "new_stage": "unaware|aware|interested|objecting|convinced|advocate", "conviction_delta": float between -0.3 and 0.3, "objection_text": "string or null", "response_text": "what you say back to {speaker_name}", "mutated_idea": "how you would describe this product to someone else", "reasoning": "your internal thought process in 1-2 sentences", "would_advocate": true|false }
prompt Recommendations prompt (once, end of simulation)
SYSTEM: You are a sharp product strategist. Be direct and specific. Return a JSON array of exactly 4 strings. No preamble. HUMAN: A product idea was simulated against {n} buyer personas. Results: - Convinced rate: {convinced_pct}% - Top objection (44%): "{top_objection}" - Second objection (29%): "{second_objection}" - Earliest adopter profile: {early_adopter_summary} - Funnel drop-off: biggest drop at {biggest_dropoff_stage} - Pitch variant winner: Variant {winner} ({winner_pct}% convinced) Give 4 specific, actionable recommendations to improve conversion. Each must name a concrete action, not generic advice. Return: ["rec1", "rec2", "rec3", "rec4"]
